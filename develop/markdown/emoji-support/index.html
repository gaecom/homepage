支持emoji表情

2021-04-05
1分钟阅读时长

可以支持各种方式的emoji表情的嵌入

在站点配置文件中[configuration] 全局启用 enableEmoji to true,然后可以在文章中直接用代码使用

<p><span class="nowrap"><span class="emojify">🙈</span> <code>🙈</code></span>  <span class="nowrap"><span class="emojify">🙉</span> <code>🙉</code></span>  <span class="nowrap"><span class="emojify">🙊</span> <code>🙊</code></span></p>
<br>

The [Emoji cheat sheet](http://www.emoji-cheat-sheet.com/) is a useful reference for emoji shorthand codes.

效果如下:

🙈 🙈 🙉 🙉 🙊 🙊


emoji表情代码参考(http://www.emoji-cheat-sheet.com/)

N.B. 你也可以通过第三方的字体包等使用表情符号

.emoji {
  font-family: Apple Color Emoji, Segoe UI Emoji, NotoColorEmoji, Segoe UI Symbol, Android Emoji, EmojiSymbols;
}
